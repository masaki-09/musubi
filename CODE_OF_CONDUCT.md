# 行動規範 / Code of Conduct

musubi プロジェクトは [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) を採用します。日本語訳は [こちら](https://www.contributor-covenant.org/ja/version/2/1/code_of_conduct/) を参照してください。

## 概要

私たちは、年齢・体格・障害の有無・民族・性別・性自認や性表現・経験のレベル・教育・社会経済的地位・国籍・容姿・人種・宗教・性的アイデンティティや指向にかかわらず、誰もが**ハラスメントのない参加体験**を提供することを誓います。

### 期待される行動

- 多様な意見・視点・経験への共感と敬意
- 建設的なフィードバックを与え、また優雅に受け入れる
- 自身の誤りに責任を持ち、影響を受けた人々に謝罪し、その経験から学ぶ
- 個人だけでなくコミュニティ全体の利益を優先する

### 容認できない行動

- 性的な言動や注目、いかなる形態の性的アプローチ
- トローリング、侮辱・中傷、個人攻撃、政治的攻撃
- 公的・私的を問わずハラスメント
- 明示的な許可なく住所・メール等の個人情報を公開すること
- その他、職業上の場で不適切と考えられる行為

## 違反の報告

行動規範違反の事例は、リポジトリ管理者 (masaki-09) まで GitHub の Issue（プライベートな内容は機微情報を含めず）または GitHub プロフィールに記載の連絡先からご報告ください。

すべての通報は速やかにかつ公正に審査・対応されます。報告者のプライバシーとセキュリティは尊重されます。
